Project Information
-------------------

**QEM (Quantitative Electron Microscopy)** is a comprehensive Python package for analyzing atomic-resolution electron microscopy images.

**Institution:** EMAT, University of Antwerp
**License:** GPL v3
**Repository:** https://github.com/zezhong-zhang/qem

Key Features
------------

- Atomic model refinement with high precision
- Multi-backend support (JAX, TensorFlow, PyTorch)
- GPU acceleration for large datasets
- Integration with materials science workflows
- Comprehensive analysis tools

Related Projects
----------------

QEM builds upon and complements several other projects:

- **StatSTEM**: The original MATLAB implementation
- **Atomap**: Python package for atomic position analysis
- **HyperSpy**: Multi-dimensional data analysis
- **ASE**: Atomic Simulation Environment
